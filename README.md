This project is a dynamic Power BI Data Analytics solution designed to track and analyze mobile phone sales performance. It provides actionable insights into sales trends, regional performance, and customer behavior to help stakeholders make data-driven decisions.

üöÄ Key Features
Executive Summary: High-level KPIs tracking Total Sales (769M), Quantity Sold (19K), and Transaction Volume (4K).

MTD (Month-To-Date) Tracking: A dedicated report view for monitoring daily sales progress against monthly targets.

Comparative Analysis: Built-in Same Period Last Year (SPLY) metrics to evaluate Year-over-Year (YoY) growth across months and quarters.

Geospatial Insights: Interactive map visualization showing sales distribution across major Indian cities like New Delhi, Mumbai, and Bangalore.

Product & Brand Performance: Deep dives into brand-specific data (Apple, Samsung, OnePlus, etc.) and specific mobile models.

Consumer Behavior: Analysis of payment methods (UPI, Card, Cash) and customer satisfaction ratings.
